Integrations

Connect Zaply to Make.com, Zapier, n8n, or your own systems.

→ Outgoing webhook

We POST to your URL whenever a customer messages you (message.received) or requests a human (handoff.requested).

{ "event": "message.received", "jid": "923001234567@s.whatsapp.net", "name": "Ali", "text": "price?", "ts": 1718000000000 }

← Inbound webhook (send messages from Make.com / Zapier)

POST to /api/v1/inbound with your API key to send a WhatsApp message to any number — perfect for appointment confirmations, order updates, etc.

POST /api/v1/inbound
X-API-Key: wak_xxxxx
{
  "to": "+17162145420",
  "message": "Hi! Your appointment is confirmed for Friday at 3pm ✅"
}

# Or use a saved template with dynamic data:
{
  "to": "+17162145420",
  "template_name": "appointment_confirm",
  "data": { "name": "Hassan", "date": "Friday 3pm" }
}

← Send API (single message)

Send a WhatsApp message from any tool using one HTTP request.

POST /api/v1/send
X-API-Key: wak_xxxxx
{ "to": "923001234567", "text": "Your order is confirmed ✅" }

⚠ Treat the API key like a password. Regenerating replaces the old key instantly.

📖 Public API — TimelinesAI-compatible endpoints

Full REST API for reading and managing chats:

GET  /api/v1/chats                    # List all chats
GET  /api/v1/chats/:jid/messages      # Chat message history
POST /api/v1/send                     # Send message
POST /api/v1/inbound                  # Inbound webhook trigger
PATCH /api/v1/chats/:jid              # Update chat (lifecycle, etc.)

🛍 E-commerce — abandoned cart recovery

Recover lost sales automatically. Send this webhook from Shopify, WooCommerce, or any store when a cart is abandoned — your customer gets a friendly WhatsApp nudge.

POST /api/v1/ecommerce/abandoned-cart

Use your API key (above) in the X-API-Key header.

💳 Payments — Stripe

Connect your Stripe account to send real payment links in chat. Without it, payment links run in demo mode.

Make.com quick start (2 minutes)

  1. Create a scenario with Webhooks → Custom webhook. Copy its URL, paste above and Save.
  2. Send a test WhatsApp message — Make captures the data structure.
  3. Add modules (Sheets, CRM, Slack…). To reply, add HTTP → Make a request with the Inbound Webhook endpoint.
